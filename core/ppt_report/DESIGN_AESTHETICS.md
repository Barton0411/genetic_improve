# PPT设计美学规范

本文档说明了育种分析PPT报告中应用的专业设计原则和标准。

## 📐 设计理念

基于 **CRAP 四大设计原则**：
- **C**ontrast (对比)：建立视觉层次
- **R**epetition (重复)：保持风格统一
- **A**lignment (对齐)：创建视觉连接
- **P**roximity (亲密性)：分组相关内容

## 🎨 配色方案

### 主题色系
- **主色调**：深蓝色 RGB(31, 73, 125) - 专业、信任、稳重
- **辅助色**：浅蓝色 RGB(68, 114, 196) - 活力、清晰
- **强调色**：暖橙色 RGB(237, 125, 49) - 突出重点

### 文字颜色层次（建立视觉层次）
1. **一级**：黑色 RGB(0, 0, 0) - 标题、最重要信息
2. **二级**：深灰 RGB(64, 64, 64) - 正文内容
3. **三级**：中灰 RGB(112, 112, 112) - 副文本
4. **四级**：浅灰 RGB(128, 128, 128) - 评论说明

### 功能色
- **成功/正向**：绿色 RGB(112, 173, 71)
- **警告**：黄色 RGB(255, 192, 0)
- **风险/危险**：红色 RGB(192, 0, 0)

## 📝 字体体系

### 字体选择
- **中文**：微软雅黑 - 商务标准，清晰易读
- **英文/数字**：Arial - 国际通用，专业规范

### 字号标准（遵循专业设计规范）

#### 封面页
- **主标题**：60pt（标准范围：54-66pt）
  - 用途：封面标题，第一视觉焦点
  - 特点：大气、震撼

- **副标题**：26pt
  - 用途：日期、场合说明
  - 对比：与主标题差34pt，形成明显层次

#### 章节页
- **章节标题**：48pt（标准范围：40-48pt）
  - 用途：章节间隔页标题
  - 特点：醒目、分隔感强

- **章节编号**：180pt（水印效果）
  - 颜色：浅灰 RGB(230, 230, 230)
  - 作用：装饰性背景元素

#### 目录页
- **目录标题**：44pt（标准范围：40-44pt）
- **编号**：36pt - 醒目、引导视线
- **项目标题**：20pt - 清晰可读
- **副标题**：14pt - 层次分明

#### 正文页
- **页面标题**：32pt（标准范围：32-36pt）
  - 用途：每页主标题

- **副标题**：20pt
  - 对比：与标题差12pt

- **正文内容**：20pt（标准范围：20-24pt）
  - 用途：段落文字、列表
  - 特点：投影可读性好

- **评论说明**：16pt
  - 用途：页面底部分析评论
  - 特点：辅助信息，层次低

#### 表格字号
- **表头**：14pt - 突出表格结构
- **数据**：12pt - 数据密集，节省空间

### 字号对比原则
- **相邻层级差值**：至少8pt以上
- **同一层级差值**：相差不超过4pt
- **最小投影字号**：24pt（保证远距离可读）

## 📏 行间距与间距

### 行间距标准
- **标题行距**：1.1倍（紧凑有力）
  - 用于：封面标题、章节标题、页面标题

- **正文行距**：1.3倍（舒适阅读）
  - 用于：段落文字、列表、评论
  - 原则：行间距约为字高的0.3倍

- **表格行距**：1.0倍（紧凑数据）
  - 用于：表格内容

### 段落间距
- **原则**：段落间距 = 行距 × 1.5-2倍
- **应用**：组间距明显大于组内行距（亲密性原则）

### 字间距
- **标题字间距**：适当缩小（0-2pt）
  - 作用：形成"区域面积"，增强整体性

- **正文字间距**：保持默认
  - 原则：保证可读性

## 🎯 布局原则

### 留白（White Space）
- **页面边距**：至少0.5-1英寸
- **元素间距**：充足呼吸空间
- **原则**：不要填满页面
- **作用**：提升视觉高级感

### 对齐（Alignment）
- **网格系统**：基于网格排列元素
- **视觉连接**：通过对齐建立元素关联
- **一致性**：整套PPT保持对齐方式统一

### 亲密性（Proximity）
- **分组原则**：相关内容靠近
- **间距控制**：
  - 组间距 > 组内行距
  - 层级差异通过间距体现

### 对比（Contrast）
- **大小对比**：字号差至少8pt
- **颜色对比**：深浅搭配，建立层次
- **位置对比**：重要信息居中或靠上
- **粗细对比**：标题加粗，正文常规

## 📊 视觉层次

### 层级控制
- **层级数量**：限制在1-3个层级
- **视觉差异**：每个层级都有明显区分

### 建立层次的手段
1. **字号大小**：主要手段，差异明显
2. **颜色深浅**：辅助手段，四级灰度
3. **字重粗细**：标题加粗，正文常规
4. **位置高低**：重要信息靠上

### 阅读顺序
- **Z型视觉路径**：左上 → 右上 → 左下 → 右下
- **F型视觉路径**：标题 → 左侧内容 → 详细信息

## 🎭 设计原则应用示例

### 封面页设计
```
标题区：60pt 深蓝色 加粗 → 震撼力
↓ (留白0.5英寸)
副标题：26pt 深灰色 → 层次感
```

### 目录页设计
```
标题：44pt 深蓝色 居中 → 引导注意
↓ (留白)
网格布局（2列4行）：
  编号：36pt 深蓝色 → 视觉锚点
  标题：20pt 黑色 加粗 → 主信息
  副标题：14pt 中灰色 → 辅助信息

组间距0.7英寸 > 组内行距 → 亲密性
```

### 正文页设计
```
页面标题：32pt 黑色 加粗 → 页面主题
↓ (留白)
内容区：20pt 深灰色 行距1.3 → 舒适阅读
↓ (留白)
评论框：16pt 浅灰色 斜体 浅色背景 → 辅助说明
```
