# PPT模板文件夹

## 使用说明

将PPT模板文件放置在此文件夹中，系统会自动识别并使用。

### 支持的文件格式
- `.pptx` (推荐)
- `.ppt`

### 文件命名建议
模板文件名应包含以下关键词之一：
- template
- 模板
- model
- 模版

例如：
- `遗传改良报告模板.pptx`
- `genetic_report_template.pptx`
- `PPT模板.pptx`

### 注意事项
1. 如果存在多个模板文件，系统将使用找到的第一个
2. 如果没有找到模板，系统将创建空白PPT
3. 模板中的现有幻灯片会被清除，只保留模板的样式和版式

### 模板查找顺序
1. 首先在项目输出文件夹中查找
2. 其次在本templates文件夹中查找
