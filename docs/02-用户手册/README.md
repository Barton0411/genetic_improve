```
1. 数据准备
   ↓
2. 数据上传
   ↓
3. 育种计算
   ↓
4. 选配推荐
   ↓
5. 报告生成
```

### 1. 数据准备
- 准备母牛基础信息Excel

## 💡 使用技巧

### 数据质量
- 使用标准模板填写数据
- 确保NAAB号格式正确
- 定期更新本地公牛数据库

### 选配优化
- 合理设置近交系数阈值
- 根据牧场实际调整策略表
- 关注库存管理

### 报告解读
- 重点关注Sheet 3（系谱识别）
- 分析Sheet 9-10（母牛指数）
- 查看Sheet 17（选配推荐结果）
