# 快速入门指南（5分钟上手）

**适用对象:** 新用户
**预计时间:** 5分钟
**更新时间:** 2025-11-02

---

## 🎯 系统简介

**伊利奶牛选配系统**是一款智能化育种管理软件，帮助牧场：
- ✅ 自动计算母牛育种值和指数
- ✅ 智能推荐最优公牛配对
- ✅ 生成专业的Excel和PPT报告
- ✅ 追踪遗传进展和近交风险

---

## 🚀 快速开始（5步）

### 第1步：启动软件 (30秒)

**Windows:**
```
双击 genetic_improve.exe
```

**macOS:**
```
双击 genetic_improve.app
```

**首次启动:**
- 系统会要求登录或注册
- 使用邀请码注册新账号
- 登录后进入主界面

---

### 第2步：创建项目 (1分钟)

1. 点击**"新建项目"**
2. 输入牧场名称（如"伊利牧场-北京"）
3. 选择项目保存位置
4. 点击**"创建"**

**项目文件夹结构:**
```
伊利牧场-北京/
├── 数据文件/     # 上传的原始数据
├── 处理结果/     # 计算结果
├── 选配结果/     # 选配方案
└── 报告/         # Excel和PPT报告
```

---

### 第3步：上传数据 (2分钟)

进入**"数据上传"**模块，按顺序上传以下数据：

| 步骤 | 数据类型 | 说明 | 必需 |
|-----|----------|------|------|
| 1 | 母牛基础信息 | Excel文件，包含母牛号、胎次、产奶等信息 | ✅ 必需 |
| 2 | 系谱信息 | 母牛的6代系谱（父、祖父、外祖父等） | ✅ 必需 |
| 3 | 备选公牛 | 计划使用的公牛清单及库存 | ✅ 必需 |
| 4 | 配种记录 | 历史配种记录（可选，用于分析） | ⭕ 可选 |

**数据模板:**
- 系统提供标准Excel模板
- 点击**"下载模板"**获取
- 按模板格式填写数据

---

### 第4步：执行选配 (1分钟)

进入**"个体选配"**模块：

1. **查看分组预览**
   - 系统自动将母牛分组（后备牛、成母牛等）
   - 勾选需要选配的组

2. **设置公牛库存**
   - 在冻精预览表中设置各公牛库存
   - 库存为0的公牛不参与选配

3. **执行选配**
   - 点击**"执行个体选配"**
   - 等待系统计算（约2-5分钟）
   - 计算完成后自动生成选配报告

**选配报告内容:**
- 每头母牛的6个选配位（性控3选 + 常规3选）
- 预测后代育种值
- 近交系数和隐性基因风险提示

---

### 第5步：生成报告 (30秒)

进入**"报告生成"**模块：

**生成Excel综合报告**
- 包含17个Sheet的详细分析
- 牧场概况、育种性状、选配结果等
- 适合详细分析和数据存档

**报告保存位置:**
```
项目文件夹/报告/
└── [牧场名称]_遗传分析综合报告_20251102.xlsx
```

**注意:** PPT自动生成功能计划在后续版本中添加（参见[项目路线图](../05-项目管理/项目路线图.md)）
