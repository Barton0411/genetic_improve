## 🎯 系统简介

**伊利奶牛选配系统**是一款专业的奶牛育种管理软件，为牧场提供：

- ✅ **智能选配**: 自动推荐最优公牛配对，控制近交和隐性基因风险
- ✅ **育种计算**: 计算母牛育种值、指数排名、系谱识别率
- ✅ **Excel报告**: 自动生成17个Sheet的综合分析报告
- ✅ **数据管理**: 云端数据库同步，本地缓存加速
- ✅ **CI/CD自动化**: GitHub Actions自动构建和发布
