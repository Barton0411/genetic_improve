# 亿速云服务器配置推荐

## 最佳方案：裸金属服务器

### 推荐配置
- **产品类型**：裸金属服务器
- **CPU**：32核心 Intel Xeon 或 AMD EPYC
- **内存**：128GB DDR4
- **存储**：1TB NVMe SSD
- **网络**：50M 独享带宽
- **操作系统**：Ubuntu 22.04 LTS
- **预估价格**：¥3,000-5,000/月

### 性能预估
- **支持并发用户**：50-70人
- **应用实例数**：10-15个
- **剩余资源**：可供模型训练使用
