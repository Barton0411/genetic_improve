# 奶牛育种近交系数与基因安全性分析工具

这个项目提供了一套计算奶牛育种近交系数和基因安全性分析的工具，帮助奶牛育种人员在配种决策时避免潜在的近交问题和遗传疾病风险。

## 功能特点

- **近交系数计算**：基于谱系信息计算配种组合的近交系数
- **基因安全性分析**：评估配种组合的隐性遗传疾病风险
- **批量分析**：支持大规模配种组合的批量分析
- **结果导出**：将分析结果导出为Excel格式，便于查看和共享
- **图形界面**：提供用户友好的图形界面进行分析操作（仅Windows版本）

## 安装指南

### 前提条件

- Python 3.7+
- pip 包管理器

### 安装步骤

1. 克隆或下载本仓库
```
git clone <仓库地址>
cd genetic_improve
```

2. 安装依赖包
```
pip install -r requirements.txt
```

3. 准备数据文件
   - 公牛数据库：默认使用`local_bull_library.db`，可在配置中指定其他路径
   - 母牛数据：需要Excel格式，包含必要的字段（见数据格式说明）

## 使用说明

### 命令行批量分析

使用命令行工具可以批量分析多组配种组合：

```bash
python batch_analysis.py --cow-file 数据/母牛数据.xlsx --bull-ids 000HO12345 000HO67890
```

#### 主要参数：

- `--cow-file`：母牛数据文件路径（必填）
- `--bull-db`：公牛数据库路径（可选，默认为`local_bull_library.db`）
- `--output`：结果输出路径（可选，默认为`results/analysis_result_YYYYMMdd_HHMMSS.xlsx`）
- `--generations`：分析的最大世代数（可选，默认为5）
- `--method`：近交系数计算方法，可选`PEDIGREE`或`GENOMIC`（默认为`PEDIGREE`）
- `--pair-file`：指定配种组合文件，需包含`cow_id`和`bull_id`两列（可选）
- `--bull-ids`：指定公牛ID列表，将与每头母牛组合分析（可选）

### 图形界面

## 数据格式说明

### 母牛数据（Excel格式）

必须包含以下字段：
- `Cow ID` 或 `cow_id`：母牛ID
- `Sire NAAB` 或 `sire_id`：母牛父系ID
- `MGS NAAB` 或 `mgs_id`：母牛母系父系ID

### 公牛数据库（SQLite格式）

必须包含以下表和字段：
- 表名：`bull_library`
- 字段：
  - `BULL NAAB`：公牛NAAB编号
  - `BULL REG`：公牛登记号
  - 各种基因位点信息（如HH1, HH2等）

## 常见问题

**Q: 如何添加新的公牛数据？**  
A: 将新公牛数据添加到SQLite数据库中。您可以使用任何SQLite管理工具进行操作。

**Q: 近交系数的安全阈值是多少？**  
A: 一般建议近交系数不超过6.25%（0.0625），但具体阈值可能因育种计划而异。

**Q: 如何解释基因安全性结果？**  
A: 
- `高风险`：双方都是携带者，有25%概率产生同合子，导致遗传疾病
- `低风险`：一方为携带者，一方为非携带者，有50%概率产生携带者后代
- `安全`：双方都是非携带者，不会产生携带者后代
- `潜在风险`：一方为携带者，另一方无数据
- `可能安全`：一方为非携带者，另一方无数据
- `未知`：双方都无数据
